# MyLouiseProject API应用

## 简介

基于Java jdk11，SpringBoot，Maven实现的轻量API应用，主要是配合<a url="https://github.com/Mrs4s/go-cqhttp">go-cqhttp</a>的接口标准实现QQ群聊机器人
